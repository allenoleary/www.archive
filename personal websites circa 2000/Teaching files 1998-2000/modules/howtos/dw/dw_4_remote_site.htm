Dreamweaver - Connecting to a remote server

If you've used another FTP program like WS_FTP this should be pretty straight forward.

First use the define sites/edit to get to the site manager window, and select Web Server Info:

What you put in each of the boxes will depend on where your server is and what information it requires for you to logon. From the top:

Server Access: Usually set to ftp (file transfer protocol)

FTP host: is the name (or number) of the server you are connecting to. The edim servers name is edim.tafe.vu.edu.au See your ISP's home page for what this should be set to for other names

Host Directory: This is the directory you want DW to set as the top level access for this site. On the edim server this is "wwwroot/" followed by your first initial and then your last name

Login: this is your user name. For the edim server it's first initial and then your last name. For other servers this is usually your email name (ie alotonic if your email address is alotronic@netspace.net.au)

Password: Your accounts password. When you use DW in a nonsecure environment (like Sth Melb or St Albans) make sure you DONT select the save option, otherwise anyone can access your site from the host computer

Use Passive FTP: This depends on the configuration of the network you're using. Try leaving it unchecked first, then check it if things aren't going right.

Use Firewall: This depends on the configuration of the network you're using. Try leaving it unchecked first, then check it if things aren't going right

Connecting

Hit OK to close this dialog and then done. Go back to the site manager window and then hit the connect button. With any luck Dreamweaver will connect to the remote site and give you another list of files in the left pane of the window. You can just drag and drop files or folders between the sites - when you drag a file with the same name onto another in the remote site, it is automatically over-written...

Use the Refresh button to update the pictures of your local and remote sites.

Getting and Putting

You can also use the get and put functions to download or upload the selected files or folders to the server, providing there is an exact match in folder structure between your local and remote sites.

Managing the remote Server

Adding folders, files or deleting is just the same as for the local site: right click and choose you option.
