Dreamweaver Exercise: Making a frameset

Read this first: Frames in Dreamweaver: an overview

Objective: To make a frameset that might be used as the basis for a standard commercial site.

1. Make a new three-pane frameset:

top
side




main

2. Colour and Save

Give each page in the framset a different background color so you can see them. Save all the parts. You might want to call the frameset document itself index.htm so it opens first.

3. Use the Frame and frameset inspectors to do the following:

  • Give all the frames names so you can target them
  • Title the frameset
  • Set the border of the frameset to 1. Colour the border so there is a line around each of the pages in the frameset
  • Set the top pane to an absolute height of 150 pixels, not resizable and to exclude scroll bars.
  • Set the side pane to an absolute width of 150 pixels, not resizable and to exclude scroll bars
  • Make the main pane freely resizable and to include scroll bars if the content is large enough
  • Check the frameset in a browser to see if it works

4. Populate the frameset.

Use pages you have already made, or make several new ones. Use targetted links to open the pages in the main window from a couple of links in the side window.

5. Publish

Send the project to your space on the edim server. Put it in it's own folder and email me the URL. (alotronic@netspace.net.au)

You have one week to complete this exercise.
