HTML resources  
     
 

sites that have guides or spec-sheets on HTML

  
     
specifications :  

www.w3c.org
The w3c is the body that looks after all HTML specifications. On their site you can get (very) detailed information about HTML and associated technologies. Also there are some detailed histories and discussion papers about things like designing for the partially-sighted.

  
'how-to's :  

www.projectcool.com
Project cool has a large range of on-line tutorials and guides. Their gettingstarted.net section is excellent.

www.gettingstarted.com
A Project Cool initiative

the webmonkey
The webmonkey (a division of the wired magazine's site) has beginners stuff, but also more on using HTML 4 and advanced techniques. Looser than the Yale guide.

Learning HTML 3.2 by Examples
This is a comprehensive how-to. Less technical than the w3c spec sheets, this guide is fast (no images) and good.

  
java and other stuff : 

www.java.sun.com
Sun are the originators of Java. This site is pretty full-on, but it's where I got the appelet we're using in class.

www.anfyjava.com
There are some fun appelets here...

earthweb.com
Earthweb is a set of very large sites that deal with just about anything that's technical on the web, from HTML to JavaScript and Visual Basic. It's quite slow from Australia, but well worth a visit... developer.com is best for JavaScript and HTML
