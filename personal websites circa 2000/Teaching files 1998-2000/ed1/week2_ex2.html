To use the image tag		  
the image tag :  

The basic image tag is

<img src="mypicture.jpg">

However is is cutomary to add width, height and an alt tag:

<img src="mypicture.jpg" width="100" height="250" alt="This is a picture of me!">

 

 
<img> attributes : 

The other useful attributes for an image are:

border="5" will put a 5 pixel border around the image in the default link color of the page. Set this to zero if the image is a link and you don't want the border to show

hspace="10" will put 10 pixels of space above and below the image

vspace="10" will put 10 pixels of space either side of the image

align="left|right|top|middle|bottom" Left and right will send the image to the next available container margin, with the text wrapping around the image. Top, Middle and Bottom do strange things with text (have a go!)

 
exercise : 

Add an image to the beginning of a paragraph of text, making sure you add the width, height and alt attributes.

Try adding borders and hspace and vspace and see how the image moves relative to the text

Add the align attribute, try left and right, then top and middle and see what happens...
