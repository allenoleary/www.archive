Behaviours in Dreamweaver

Getting New Ones

Step 1: Go to the Dreamweaver Developers Zone

Make sure you're online (!) then, from the help menu, select the Dreamweaver Exchange

Step 2: Download and install the new Extensions Manager

They didn't get it right the first time, so this is an essential download.

Step3: Go back to the developers center and login

You have to login to get the goodies. If you have objections to that or have difficulties, there are plenty of sites to choose extensions from. Macromedia have just put the best/most reliable ones in a central spot. Try www.masimocorner.com or www.yaromat.com if you can't login...

Step4: Have a look through the behaviours + pick one

Lots and lots for various things. Some are new objects (like a beatnik object), some are behaviours as such. Pick one you like the look of and download it to your desktop

Step 5: Open the Extensions Manager

Select it from the Windows menu. Select add new and surf to the file you just downloaded to your desktop. Bingo

Step 6: Use it...

Depending on whether you downloaded an object or a behaviour, it should appear in the appropriate menu straight away...
