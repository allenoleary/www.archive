Extending Tables :

Recreate, as closely as possible, this contents page from Spark Online, August 2000. Use tables to manage color, layout images and control text.

Brief:

1. Download the images off the spark page.

2. Recreate the table structure:

You will need two tables: one for the top navigation bar and another to control the left strip and the page contents.

None of the links need to work! This is purely a formatting exercise...

Tips:

  • You might need a transperant image to control the width of the left and right sides of the table.
  • Both the top and the main table will need to be absolutely sized to the same width as the very top image
  • To get the tables to bump right together under each other, just finish one table and start another without any <br> or <p> tags.
  • Use the bgcolor of the table cell to set the color of the cell (and feel free to play with colors as you go, it don;t have to be all grey)
  • Put a non-breaking space (&nbsp;) into the table cells to make it render when there is no actual content for the row
  • Looking at the source of the page won't be much good to you, your page should be simpler...

3. Align the table to the left and top of the browser window with no browser offset

Remember to put marginheight=0 marginwidth=0 leftmargin=0 topmargin=0 into the <body> tag to get rid of all browser offset

4. Don't get bound up in knots!

Spend 3-4 hours MAX on this. You should be able to get the top section working, and have a good go at the body section...

5. Publish to the server into it's own folder.
