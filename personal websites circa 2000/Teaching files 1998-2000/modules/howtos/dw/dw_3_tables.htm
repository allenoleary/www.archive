Dreamweaver - dealing with tables

After the laborious effort of hand-coding tables, Dreamweaver is a dream indeed..

A brief run down:

  • Inserting Tables

    Use the button to get to the tables dialog box

  • Selecting tables and controlling table properties:


Click the table edge to select the whole table and get the above property inspector.

W sets the Table width, select % or pixels from the drop-down to specify absolute or relative width. Likewise H (not reliable across all browsers).

Most of the attributes available to you from this palette should be familiar to you. In the bottom right of the palette Bg sets the background image for a table (not reliable across all browsers) while Bg is table background color and Brdr is the border color for the table.

  • Inserting and Deleting rows and columns

    Right Click inside the table and select table from the drop-down, there's a range of useful functions there including inserting single of multiple rows or columns

  • Formatting cell alignment and contents

This is the default font inspector, changes will apply to the currently selected cell or cells. Horz and Vert set the align property of the cell. Bg and Brdr as above. No Wrap forces content in that cell to have no line breaking...

  • Merging and Splitting Cells

    To merge cells, select adjacent cells and use the right-click menu to select Merge Cells

  • Heigth and Width, not all they seem...

While it may be a nice idea to be able to drag table row and column borders around, it's not always so simple. Some browsers won't repect your table ideas at all. First thing to remember is that a table will always expand to fit whatever is inside it - even if you have set it to an absolute width.

The only absolute way to lock a table off to to create what's called a buttressed table, which can be created in Fireworks or Image Ready.
