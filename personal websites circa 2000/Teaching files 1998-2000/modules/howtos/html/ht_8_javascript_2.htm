Using the Date() Object

JavaScript can interrogate the computer system and find out things like the local time. Try altering the Alert example we were using last week to put a date into the alert box:

<a href="#" onCLick="window.alert('It is now ' + Date())">What's the time Mr Wolf?</a>

Whats the time mr Wolf?

Using the Document.Write() method

Likewise we can use the document.write method to write html based on information like the above:

<script language="JavaScript">
<!--
document.write('This file was loaded into your browser at ' + Date())
//-->
</script>

Which looks like this:

(Try refreshing your browser to see it change)

Note that this sort of JavaScript is written into the body of the html document, and is not attached to any events. It just runs as the page is downloaded...
