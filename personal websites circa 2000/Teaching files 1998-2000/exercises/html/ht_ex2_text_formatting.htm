<b>Bold text</b>

<blockquote>This paragraph will be intented both sides from its container</blockquote>

The Font Tag

<font family="verdana">Changes the text to Verdana - if the user has the font installed on their system!</font>

<font size="2">Changes the size of the font to two - the default is three, as above. Note you can also use relative values "-1"or "+1"as well</font>

<font color="#0066FF">Colors the text to the appropriate hex values</font>

<font family="verdana, arial, sans-serif" size="2" color="#0066FF>This tag does all three at once</font>

Lists

Use 'view source' to see if you can work out the tags for these

A basic Ordered list:

  1. One
  2. Two
  3. Three
  4. Four

A basic unordered list:

  • One
  • Two
  • Three
  • Four

Note that you can 'nest'lists as well:

  1. One
  2. Two
  3. Three
    • One
    • Two
    • Three
    • Four
  4. Four

Or start them in different points with different characters

  1. One
  2. Two
  3. Three
  4. Four
  • One
  • Two
  • Three
  • Four
