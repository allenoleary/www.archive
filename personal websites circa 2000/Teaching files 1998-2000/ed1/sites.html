Sites to See  
     
 

a collection of sites that demonstrate concepts from class

  
     
week 1 : 
  

www.glasswings.com.au
This site is one of the oldest in australia, and would probably run on a cell-phone. Katherine, the owner, is meticulous with her hand-coding.

www.ctheory.com
Another very old site. This one is all about the content, presentation is not important here.

www.slv.vic.gov.au
Victoria's State Library. A clean site that presents information in a logical and rigorous manner.

www.yahoo.com
A `portal' site that has been stripped of all extraneous material in an effort to be fast-loading.

www.looksmart.com.au
A directory site that has been made as easy as possible to use with it's cascading menus. There are hunderds of thousands of sites listed here.

www.manutd.com
A no holds-barred sports site with information, virtual tours and, of course, an opportunity to buy product.

http://www.sciarc.edu/
A fine example of an html site.

http://www.designiskinky.net
An Australian web design hangout - lots of fantastic links to get you thinking...

 
week 2 :
 

www.roxx.com
This is a designers skite site that makes really good use of frames for design effect. Also has excellent links...

Lynda Weinmanns Site
This is where that color chart that you might want is...

altas magazine
A sophisticated use of tables to position contents and `slice' graphics (watch this page load to see what I mean `slice')

Note how they use background images to good effect...

juliet martin's
Here's a site that uses very simple html and text to play around with the users perception of presentation... no tables!

RGA design
Here's a design site that presents information, but in a very assertive and colorful manner, making a mundane task (checking out someone's web site) into a experience...


 
week 3 : 
 

earthweb.com
Earthweb is a set of very large sites that deal with just about anything that's technical on the web, from HTML to JavaScript and Visual Basic. It's quite slow from Australia, but well worth a visit... developer.com is best for JavaScript and HTML

An improved Color Maker...
I found this one at Netscapes' `dev edge' site...

 

  
week 4 : 

javascripts.com
One of very many javaScript resources. Try Netscape too, especially their devEdge area.

htmlgoodies.com
A nice site with lots of stuff to help you understand html and javaScript

developer.com
Devloper has more resources than you could ever possibly want for anything web related. Not very beginner-oriented, but still worth a look.
