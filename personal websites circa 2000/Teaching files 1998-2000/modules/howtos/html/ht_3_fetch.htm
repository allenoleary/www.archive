Using Fetch

Publishing to the web means transferring your html and image files form your local hand drive or zip to your folder on a 'live' web server. Fetch is a (very) rudimentary tool for the mac. On the PC you would use CuteFTP 4 or WS_FTP LE both of which are free downloads.

To accomplish this you need to know several things:

  • Where your local files are
  • What the remote servers ftp details are:

    A host name
    A initial or root folder for your site
    Your user name
    Your password

1. Start fetch and enter your posting details:

For you the details are as follows:

host: edim.tafe.vu.edu.au
root folder: wwwroot/aoleary (your first initial plus your last name)
user name: aoleary (your first initial plus your last name)
your password: whatever you wrote in your sheet.

Hit Ok to connect

2. Surf to (or create) the folder you want to put your files to on the remote server

3. Select the files (or folder) you want to publish

4. Hit fetch to make it happen

5. Check your files on the server by typing into a browser:

http://edim.tafe.edu.au/aoleary/foldername/ (or whatever your user name is...)

*this will only work if you have a file called index.htm, otherwise you would have to type the fullpath of the file.
