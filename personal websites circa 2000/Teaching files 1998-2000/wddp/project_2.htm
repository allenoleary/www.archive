Advanced Web Development Programme

Project 2: Build the Website you scoped in Project 1

Due: Week 7 (last class)

Aims:

This project is designed to make you build and publish the website that you have been planning, incorporating the publishing skills you have been learning in Dreamweaver and Fireworks:

  • Use Dreamweaver and Fireworks to realise the website you have planned

  • Deploy web techniques that you have learnt in this course (and the previous) such as CSS, javascript and dhtml, using Dreamweaver as your publishing tool

  • Design, realise and publish pages using Fireworks features

  • Publish and check your sites on both platforms and browsers

Tasks:

To satisfy the demands of the project you will need to complete the following tasks:

  • Use Dreamweaver and your knowledge of html to control framesets, tables and text formatting, including CSS.

  • Apply dreamweaver to the creation of dhtml features such as layers, rollovers and other advanced behaviors

  • Develop and make effective navigation schemes

  • Use Fireworks to optimise images, make rollovers, image-swaps and disjointed rollovers

  • Use Dreamweavers site-manager functions to check your site for errors and then publish to the edim server

  • Test your live site on different platforms and browsers and trouble-shoot

  • Briefly present your site to the class

Presentation:

This project will be published to the edim server.

Assessment:

This site will be assessed on both platforms and in IE4 and NS4. I won't use version three browsers. I will view the site from home to realistically assess download times and performance. I will view your site at various resolutions, but take into account your user profile when marking.

Length: I would expect that this site will be around 20 pages of content, plus navigation pages

In addition I will be looking for the following things:

A sense of style and purpose with your design
A sure targeting of your user group
An understanding of the paths your user can take through your site
An adventerous and effective use of the more esoteric dhtml features
And, as always, a touch of flair...

Requirements:

To get a good mark I will expect you to use most of the following technologies:

Frames - not compulsory, but suggested
Tables - I would expect you to use tables extensively to control page formatting
Forms - While we are not using forms extensively in this block, using a jump menu or similar would be nice
CSS - Use an external style sheet to at least cover basic text formatting
Javascript - Use standard javascripts like image rollovers
Dhtml - Use several of Dreamweavers behaviors in your project. Maybe include a timeline animation on your opening page. Better still, find some more you might use at the Dreamweaver developers center
