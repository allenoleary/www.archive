Frames in Dreamweaver

Frames in Dreamweaver are still a little convoluted (as they are in general!) The main difficulties are understanding how many files you have open at once, how to save them and how to select the various frame and frameset properties.

How may fingers am I showing?

Remember how hand-coded frames work: The host file (frameset file) holds the content files and is a seperate html document. It's no different in Dreamweaver, but you don't see the frameset file in the environment. The following illustration indicates this. Here there are currently FOUR files open at once; three content pages and the frameset page itself.


To work with frames properly you need to have the Propertes inspector and the Frames palette open (from the Window menu). Unlike other html elements, it is easiest to select the various options for the framset an frames by using the palette and not clicking in the workspace.

Creating a Frameset in Dreamweaver

  • Open a new page.
  • Use the Modify/Framesets menu to split the page into parts
  • Open the frames palette to see the layout of the pages

Saving the files and frameset itself.

The important thing to remember is that saving the contetn pages does not save the frameset page, and vice versa.

  • Click inside each page in the main window. Select save and save each open page into your root folder.
  • To save the configuration of the frameset (the file you can't see) Select Save Frameset from the file menu. Remember: everytime you save the frameset you save the current configuration of the frameset, but not the contetn pages inside the frameset
  • Check you've got everything right by opening the frameset file in a browser
  • You can also open existing pages into a frameset by Using Open in Frame from the file menu.

When you F12 to preview the frameset, you will often be asked if you want to save all files before continuing. This can get you into trouble because you are saving the current configuration, even if you are working on different pages than will appear in the final frameset.

Setting Options for the <frameset>

Use the frames palette.

Selecting the outside border of the frameset lets you set <frameset> properties:


Gives you the property inspector:


From here you can select options for the borders of the frameset, and also the dimensions of each of the panes from the right-hand side of the inspector. You can use pixels, percentages or relative measures (put a * in the relative field to free-size a frame)

While you have the frameset selected, click on the html source button to see the framset document itself - and while you're there, give this page a title, it's what will appear in the browser window. Also while you're here you can add <noframes> content.

Selecting the <frame> options

This is where we set the page offsets, individual names for the frames, and scrolling properties for each page.

First click inside a frame:


The property inspector changes to give you the frame options:

From here give each window a FrameName (this becomes the target attribute in your link), set scrolling, whether the frame can be dragged (No Resize disables dragging), page offset (margin height and width) and further border colouring if you're that keen!

Make changes, and again select the frameset and hit html source. See that DW has added information inside the individual <frame> tag.

Tips and Tricks:

  • As soon as you have created your frameset, save it and all the pages.
  • Use some kind of logical naming for your files and stick to it
  • So set invisible borders, set all the options in the <frameset> and <frame> tags to "no" and 0, just to be sure
  • Dreamweaver doesn't display border colors properly: you need to open the framset in a browser to see them work properly.
