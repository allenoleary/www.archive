Text-only version

This area of the alotronic website has been specifically designed for maximum user access. In the work I have done in the web publishing corporate sector, this is not usually a priority. However, the W3C has laid out clear guidelines fo ensuring maximum access and a system of ranking websites for access issues. this system is entirely voluntary.

In general, those designing sites for maximum access should consider the following points:

  1. Users may not have the latest hardware or software.
  2. Users may have a slow internet connection, or pay a premium for access.
  3. Users may have a visual or other disablity.

The W3C has many solutions for the above problems which fall in several areas:

  1. Provide text alternatives for any images (including animations) through 'alt' tags, 'longdesc' tags and 'title' tags.
  2. Provide layout and style information through style-sheets and not 'font' tags.
  3. Provide alternative navigation for users using 'tabindex' or 'accesskey' attributes, and give more information about link targets

I have incorporated all these features into this area of this site.

Xhtml

This site is also ready for the future, all pages are XHTML compliant. XHTML is the inbetween markup between HTML and XML. It has no DTD like XML, but can at least be read by an XML parser without going bellyup.
