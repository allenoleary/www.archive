Imaging for the Web

    A brief history of the two common formats: gif and jpeg

    Structure of the file types and uses

    gifs:

    • max 256 colors
    • support transperancy
    • animation
    • lossless compression
    • uses indexed color palettes to select colors from original image
    • interlacing ability

     

    jpegs:

    • max 16.7 million colors
    • lossy compression (throws picture information away)
    • compresses on a percentage basic
    • progressive ability
