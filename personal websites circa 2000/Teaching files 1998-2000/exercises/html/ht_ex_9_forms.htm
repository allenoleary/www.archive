HTML Exercise: Forms

Format and Publish to the Server a Fake User Information and Feedback Form for an imaginary company, service or organisation using each of the form elements we've just discussed:

  • Textbox
  • Textarea
  • Password
  • A group of CheckBoxs
  • A group of Radio Buttons
  • List Box
  • A Submit and Reset Button
  • Set Action to an imaginary URL and Method to Post
