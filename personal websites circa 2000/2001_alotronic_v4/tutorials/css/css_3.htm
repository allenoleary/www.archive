Style Sheets 3: Three Ways of formatting style information

The Idea

CSS has three ways of inserting tyle information, at tag level, embedded style sheet and external or linked style sheet level

Tag Level

At this level you just write the style information into the tag itself:

<P style="color:#44444; left-margin:100px; right-margin:100px">

THis is a good quick way of applying a style to a single element, but has the disadvantage of not being applied document wide, which means lots of code editing to make changes.

Embedded STyle SHeet

This is what we did last week, where the style declaration sits in the head of the document:

<html>
<head>
<title>Style Sheet Example</title>
<style type="text/css">
<!--
body { background-color: #FFFFCC; margin-top: 18px; margin-right: 100px; margin-left: 100px}
p { font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 12px; color: #000033; line-height: 150%}
h1 { font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 20px; font-weight: bold; color: #000033}
-->
</style>
</head>

<body bgcolor="#FFFFFF">

<h1>This is a normal heading</h1>
<p>This is just paragraph text and shows how the margin for the whole page is controlled as well as how the line-height has been altered as well to 150%</p>
<p>This is another paragraph.</p>

</body>
</html>

Linked style sheets

THis is where we store the style sheet as a text file and link to it. THis means that we can update a whole site at once, just by changing one text document.

Here's how:

  1. From last week cut the style declaration out of one of your documents (everything between the <style> tags)

  2. Paste the text into a new simpletext document.

  3. Save the new document with a name and the .css extention

  4. In your orginal document you need to place the following code inbetween the <head> tags:
    <link rel="stylesheet" href="filename.css">

This will force the browser to load the one style sheet for every document you put this line of code in. Making changes to the style sheet will affect all the files that link to it.

More formatting; the custom class and the <span> tag.

When you want to make a style declaration for something out of the ordinary, or to make a custum format for a special set of formatting (like a 'copyright' notice) you can make a new class.

In your style sheet you make a new class by adding the .classname declaration (where classname can be anything you like). For instance:

<style type="text/css">
<!--
p { color:#FFFFFF; font-size: medium }
.red {color:#CC0066}
.small {font-size:small }
//-->
</style>

This means that you can make any text "red" by using the span tag:

<P>THis text is the paragraph color<span class="red"> but this bit is red</span> and this is normal again.</P>

You can also apply a class to a whole paragraph at once:

<p class="red">Now all the text will be red</p>

This power to format a whole paragraph at once means you can make special paragraphs for, say, those copyright information blocks you sometimes see at the bottom of a page:

.copyright { font-size: smallest; text-align: center; border: solid thin #CC3366 }

Then apply the style to a paragraph:

<p class="copyright">This document remains the property of me (sure...)</p>

The Pseudo Classes

This is just a fancy way of talking about specifying the link, vlink, alink properties:

A:link {color: blue; text-decoration:none}
A:active {color: red; text-decoration:none}
A:visited {color: black; text-decoration:none}

There's also 'A:hover' which changes the link when you roll over it (a roll-over without having to use images). Also note that using 'text-decoration: none' will remove the underline from your links.
