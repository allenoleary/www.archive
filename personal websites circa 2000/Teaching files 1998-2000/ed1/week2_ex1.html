to understand the dirrerence between the three types of links:

  • local
  • relative and
  • absolute

 
local links :  

Use local links to move up and down a single page (the page has to be deeper than the brpwser window for this to work!)

The anchor for the link is defined like this:

<a name="topofpage"></a>

To link to that anchor we use the following code:

<a href="#topofpage">go to the top</a>

Note the "#" in the link...

  
relative links : 

Use relative links to move around folders and pages that are in the same website as the host page.

To link to a page in the same folder:

<a href="nextpage.htm">go to next...</a>

To link to a page in a folder UNDER the host page:

<a href="foldername/nextpage.htm">go to next...</a>

To link to a page in a folder ABOVE the host page:

<a href="../nextpage.htm">go to next...</a>

 

  
absolute links : 

Use absolute links to go to other websites by using the full path of the target page:

<a href="http://www.mysite.com">Go!</a>

If you add target="_blank" as an attribute the page will load in a new browser window.
