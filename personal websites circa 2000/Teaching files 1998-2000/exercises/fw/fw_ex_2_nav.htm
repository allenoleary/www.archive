Dreamweaver/fireworks Exercise: Making advanced Image Swaps and Nav Bars

As well as making simple rollovers with Fireworks, we can also do more intriguing things by using the "swap image" and the "Set Nav Bar" behaviours.

Swap Image

Here we can make a "disjointed rollover" by selecting another (or many other) slices to swap when the current slice is rolled over.

Make an image and get the slices in, then in the behaviours palette click the "swap image" behaviour. Check out the dialog box...

For each image you want to swap, click on that slice, and then choose where you want to get the swapped image from. This can be from another file on the site, but the best place to get them is from frames in your image (you can gaurantee that they're the right size then!)

So... make some swap images on new frames, and make them happen.

You can check that they're all working by selecting the preview tab on the image palette.

Nav Bars

Nav bars are collections of images that use JavaScript to track whether an image has been clicked or not. If it has been clicked, then it stays in the highlighted state (frame 3) until another is clicked. In the meantime, all the other rollovers keep functioning.

Exercise

1. Make a new image in fireworks, this will be used as a navigation bar in a frameset.

2. Make four rollovers buttons (frame 1 and 2) as per last week. Check they work.

3. Add a third frame for the Nav Bar and change the buttons accordingly (this frame will contain the button states that will occur when the user clicks on the image)

4. Go back to the slices and change their behaviours from "Simple rollover" to "Set Nav Bar". Leave both options UNselected

5. Test the results again, clicking on the images as well as rolling over them.

Part Two: Swap Image, or how to provide more information about a rollover

1. Adjust the slices to make room for a single slice that will change text as each button is rolled over, describing more about a section. This has to be away from the rollover slices.

2. Make as many new frames as you have rollovers (in this case four, so you will have 7 frames altogether)

3. In the position of the new slice, add a different piece of text in frames 4 through 7

4. Click each of the slices and add a swap-image behaviour, choosing the relevant frame from your document in turn.

5. Preview to see if it works.

6. If it does, optimise the slices, add alt text information and publish the images and page to the server and email me the url for assesment!
