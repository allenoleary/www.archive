Fireworks - Basic Image Optimisation

In Fireworks you use the optimise Palette to set the export defaults. The first thing to decide is whether you want to export to gif or jpeg:

gifs: good for crisp text and solid areas of color

  • max 256 colors
  • support transparency
  • animation
  • lossless compression
  • uses indexed color palettes to select colors from original image
  • interlacing ability

jpegs: good for photographic images

  • max 16.7 million colors
  • lossy compression (throws picture information away)
  • compresses on a percentage basic
  • progressive ability

If you aren't happy with what this means, you should step through this tutorial from webuilder.com:

Webuilder Imaging topic (about 8 pages)

The JPEG options:

Settings: Selects the basic options and settings (you can save a particular setup you like as well).

Quality: The most important setting. 80% will reduce the file size by over half and be almost indistinguishable from a file exported at 100%. This is the most common setting for photographic images on the web.

Smoothing: This setting will blur the image slightly. Useful at low % settings, depending on the make up of the image

The GIF options

The gif palette is a little more complicated. You have to make decisions about which of the possible 256 colors a gif will hold that you want to use. The sub-selection box under the GIF setting allows finer control:

Web 216: Currently highlighted is Web 216 which forces all colors in the image to the web safe palette

Exact: leaves values as they are, but chooses the most-used 256.

Adaptive: Chooses the best 256 colors in an image. Produces the best results, but there's a danger that the image will look bad on a low-quality system

Web Safe Adaptive: This snaps colors to web safe if they are close in the color table, otherwise leaves them alone. This is a good compromise as the snapped colors will be consistent across all browsers and platforms.

Colors: Lets you set the maximum number of colors in the exported palette (reducing the colors reduces file size)

Dither: Sets dither options for the image - dithering will 'mix' the colors you have chosen

Transperancy

The small eyedropper tools along the bottom of the palette let you add and subtract transparency to an image. To use the transparency of the image as it exists in Fireworks, select alpha transparency Index transparency uses the canvas color and marks that as transparent.

Using 2up and 4up

The 2up and 4up tab in the image palette lets you compare various settings on the current image. This lets you apply settings to an instance of the image to compare your settings.

Notice that there is an image size and download time read out in the bottom of the window.

You can use the hand and zoom tool to move around the image and get a closer look

When you have picked the setting you want, select that instance of the image and select file/export (crtl-shift R).

Using Export Preview

Export Preview ( the the file menu) gives you greater control over the image, and lets you export the image to another size:

 

As well as all the above options, clicking on the file tab will let you export as a percentage or to another pixel size.

Down the bottom left of the dialog are a couple of useful extra features:

The wand starts the export wizard which steps you through the whole process

The k with the vice clamp around it will optimise the image to a target size. Click it and type a k value into the popup to set the target size.
