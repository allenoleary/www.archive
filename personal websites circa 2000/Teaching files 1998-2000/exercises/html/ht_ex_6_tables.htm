Tables to layout pages:

Tables have become the way that designers have come to layout pages, they're the closest we get to being able to use a layout grid.

Analyse one of the following sites - or any table based site: Draw out on piece of paper the table layout you think they might have used... Pay particular attention as the page loads - sometimes you can see the frame structure.

http://home.cnet.com/webbuilding/
http://www.abc.net.au/default.htm

After you've sketched it out, save the page off the internet and open it in a text editor to see how they did it.

Review your Blue project:

How would you use tables to improve the look of your Blur/Red project pages?

Draw up a table layout, then see if you can make a page that works the way you want it to.
