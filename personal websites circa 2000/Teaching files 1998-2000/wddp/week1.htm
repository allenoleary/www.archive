    Exercise: set up a local root folder for an exisiting site

    1. First, make a copy of your site!!
    2. Select 'new site...' from the Site menu
    3. In the 'local info' area, give the site a name and select the topmost folder of the site
    4. Click Ok, view the site using the 'Site Files...' option (F5 to toggles document and site view)

Text exercise: build a basic page... (I'll be looking at this one)

Open a new document.

Add a paragraph or two of text and an image. Format the paragraph text in a number of styles using the property inspector.

Make a list, try indenting and outdenting a paragraph

Insert the following

  • a horizontal rule
  • a copyright symbol
  • the date
  • an html comment
  • a few non-breaking spaces
  • a <br> tag
  • an email link, an absolute link to a website
  • a <meta> tag listing some keywords for this page.

Make an html style that will change a whole paragraph to size 2 dark blue verdana, make an html style that will change a selection to bold and red.

Don't forget to check your spelling!

If you get through this quickly, open a html page you have already written by hand and see what it looks like in Dreamweaver.
