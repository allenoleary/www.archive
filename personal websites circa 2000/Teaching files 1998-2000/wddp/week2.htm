Fireworks: The Application and uses, basic image optimisation

Fireworks is Macromedias specialised web graphics application. It has a number of features that make it ideal for making, optimising and exporting web graphics:

  • Works with both bitmaps and vector formats at once

  • Allows multimple non-destructive styles and effects to be added to any object

  • Has special optimisation and web export features

  • Will write complex html and javascript to allow you to make complex 'buttressed' tables

  • Extensive search and replace features (text, color, styles)

  • A symbol/instance feature similar to Flash

  • Excellent for creating animated gifs

This week we'll just have a quick look around the environment and learn basic image optimisation techniques

Design Topic: Introduction to information design and site architecture

Information design is nothing new. The Novel or enclyclopedia have very resolved and conventional information designs. (We'll have a look at those in class)

New Media is beginning to have some conventions of it's own (top banner, navigation on the left-hand side).

There is also a growing cross-over between the web and other information rich environments like a newspaper. We'll take a look at that, and get the information order/visual order idea in our heads.
