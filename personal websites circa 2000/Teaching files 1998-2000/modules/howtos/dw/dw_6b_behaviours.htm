Behaviours in Dreamweaver

So on to how to do it in Dreamweaver:

Step 1: Choose an html entity

Because Javascript (mostly) uses events on html elements to catch actions, first you need to select the html entity to attach the behaviour to. This means just clicking on an image or selecting the text you want.

Step 2: Attach a null link

In the link box in the property inspector, insert a #. If you want to actually put a link there. do that instead.

Step3: Select the html entity again

Click on the image or inside the text you have just assigned the null-link to.

Step4: Display the behviours palette

Select windows/behaviours to get it

Step 5: Select the behaviour

Click and hold the little + button and the available behaviours for the type of object you have selected comes up.

Step 6: enter any information the behaviour requires

This will vary depending on the behaviour

Step 7: Select an event you want to trigger the action

From the drop-down box. Make sure that you have the appropriate Events For: option selected (I suggest ver 4 and later browsers for maximum compatibility)

Step 8: Preview your page and see if it works

What each of the behaviours does:

Call javascript: here you can type any little scrap of javascript such as window.close() to close the current window - this is the same as writing it in the <a href> tag

Change property: this lets you change propeties of layers (like positions etc) but you have to know what you're doing...

Check Browser: This will set up a browser sniffer and redirect a user to various other pages depending on the result. Useful to detect if the user is using an old browser version when they enter your site

Check Plugin: Same as check browser but collects information about the users plugin capabilities

Control Shcokwave or flash: lets you play, stop, jump around flash or shockwave moview in the page

Drag layer: Lets you select various layers and make them draggable. Also constrains movement and triggers javaScript when you drop it

Go to URL: does exactly that. Use this when you want to load two or more pages in a frameset with one click

Jump Menu: Inserts a javaScript menu which will redirect the browser from choices made in a drop-down list

Open Browser Window: Very Useful, this one lets you open a new window and gives you control over size and tollbars etc

Play Sound. Does that. A little unreliable.

Popup Message: Just an alert box

Preload Images: You can manually preload images with this. rollovers are usually automatically preloaded anyway

Set nav Bar Image: works with the Insert navigation bar object.

Set Text: Gives you control over various place to put text. Status Bar is most useful

Show Hide Layers: Lets you switch on/off layers in your document. Useful to add extra information with a roll-over (like a tool-tip)

Swap Image: yes.

Swap Image Restore: yes.

Timeline: Allows you to control any timelines you have in your page

Validate Form: Lets you test form fields for types of content such as a number or email link.

Get More Behaviours: Takes you to the Dreamweaver Exchange site to get more behaviours.
