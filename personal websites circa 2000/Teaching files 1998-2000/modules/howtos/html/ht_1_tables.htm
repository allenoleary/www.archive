HTML Tables

Tables

Tables are the most used layout feature of html. While tables were initally meant to serve strictly tabular information, they soon became stock-in-trade of designers as a way of laying out all information on a page, usually by making them invisible.

First we'll look at tables in their traditional incarnation, and then as a layout tool.

The Basic Table, with Header Cells.

With this table also note a couple of things:

The nowrap attribute forces the table cell to forgo any line breaking (this will often blow a table out)
The image in the background of the bottom left cell: It's tiled like a page background image - doesn't effect the size of the table cell.

Tables as a layout Tool

Since tables are as close as html gets to a layout grid, they are used extensively to format pages. Because they can be nested inside each other, you can achieve many looks (with lot's of patience). A very common way to use tables is to shove everything in a page inside a table that is fixed width (like this page here) or with width set to a hundred percent.
