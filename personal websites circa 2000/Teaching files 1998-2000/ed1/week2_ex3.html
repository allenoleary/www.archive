basic table :  

The basic basic table looks like this:

<table>

<tr>

<td>Cell One</td>
<td>Cell Two</td>

</tr>
<tr>

<td>Cell Three</td>
<td>Cell Four</td>

<tr>

</table>

 

 
<table> attributes : 

The other useful attributes for an table are:

width="600px" sizes the Table to an absolute width

width="80%" sizes a table to a relative width of 80% of the browser window

align=left|center|right

bgcolor="#EEEEEE" sets the backgroud color of the whole table

background="imagename.jpg" Tiles an image in the background of the whole table.

border="5" will render a border 5 pixels wide - "0" will turn off all borders

bordercolor="#003399" will color the border

  
<td> tags :

Use the following attributes inside individual <td> tages to change the appearence and alignment of the individual cell:

bgcolor="#EEEEEEE" as above, but just this cell

background="myimage.jpg" as above, but just this cell

align="left|center|right"

valign="top|middle|bottom"

To colour the text in a cell, use the <font> tag
