Linking

Exercise: Include one of each kind of link in page (you will need to make another local html page to link to...)

Absolute links:

This code:

<p><a href="http://www.apple.com">This text is underlined</a></p>

Gives you:

This text is underlined which will link to the apple website

 

Relative Links

This code

<p><a href="week2.htm">Go back to the week two page.</a> Links to a page in the same folder as the calling page</p>

Gives you:

Go back to the week two page. Links to a page in the same folder as the calling page

 

Local links

This code

<p><a href="#top">Top of page</a>

Gives you

Top of page. This tag will like to a 'local anchor' <a name="top></a> within the calling page

 

Email links

This code

<a href="mailto:alotronic@netspace.net.au">Email me</a>

Gives you

Email me When the user clicks this their default email client will open with the address field filled in.
