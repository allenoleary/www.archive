HTML Exercise: Style Sheets

This will be a class exercise that will count towards your class-work mark.

The Basic idea

Get this page and reformat - and improve - it using style sheets. You don't have to acheive exactly the same look, I'd even prefer it if you added a touch of design sensibility to it...

Requirements:

  • Use an external (linked) style sheet.
  • Redefine all the html tags present on the page (including the body tag)
  • Make custom classes for the author, copyright and navigation text.
  • If you're feeling clever, try taking the text out of table format and use the margin, border and padding attributes to give it a similar look.
