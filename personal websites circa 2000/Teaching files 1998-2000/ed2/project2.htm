Electronic Design 2

Tutor: Allen O'Leary (alotronic@netspace.net.au)

Project Two: Print to Web (20% of final Assesment)

Due: On server 10th October (2 weeks)

Aims :  

To repurpose print material to the web, considering both design and html issues.

Brief:

To make a website that repurposes print material into a website.

You can use any Print material you wish, but bear in mind the following points:

  • The finished site should be 8 to 12 pages
  • None of the pages has too big, but you should be presenting a relevant & logical amount of information on each page.
  • You will need to scan and treat images for the web from your print material
  • Choose some print material that you are actually interested in (I've got some examples in class)
  • You will need to carefully consider how you are going to redesign the navigation and site structure for the web

Timeline:  

This project is due on the server on the 10th of October. No informal extensions will be granted for this exercise.

There will be time to work on this in class in the next two weeks, an hour a week.

If you get stuck for ideas, have a look at http://www.coolhomepages.com for ideas

Requirements:  

  • Project due on server.
  • Make 8-12 pages
  • You will need to give me the print material as well, so can see what you've done
  • Use style sheets to control text formatting at least
  • Framesets are optional, as are tables, but both might make good sense!
  • Use metatags on each page as if you were going to attract search engines

  • It would be good to augment your pages with basic rollovers, but this is not compulsory

Important points:  

    • Remember to name you files in lower case and leave no spaces in file names.
    • Make use of folders to seperate your images and html pages
    • Name the first page of your site index.htm so I know where to begin and link to it from your homepage

Assesment:  

I will be looking for the following things:

  • Correct use of html, Css, javaScript etc
  • A lot of creative flair in exploring the conceptual shift from print to screen
