Style Sheets 2: Margins, Borders and Positioning.

The Idea

CSS has a lot more power to manipulate margins, borders and positioning than plain html. These declarations are called 'Box properties'.

Margins

For instance on the previous page I used this in the body declaration to control the page margins:

margin-top: 20px; margin-right: 100px; margin-left: 100px

I could comfortably use relative measures as well:

margin-top: 5%; margin-right: 10%; margin-left: 10%

Because I put these measures in the body declaration, they will apply to everything inbetween the <body> tags. There is no reason why I couldn't put this information into the P or H1 declarations instead, and I could do this to outdent all H1 headings from paragraph text:

h1 { margin-left: 75px; margin-right: 75px}
p {margin-left: 100px; margin-right: 100px)

If I declare margins in the body and in the P and H1, they will be added together because the P and H1 are child elements of the B parent.

So margins are used to position content within their parent elements:

'margin-top', 'margin-right', 'margin-bottom', 'margin-left'

Value

These property sets the specified margin of an element:

'margin'

The 'margin' property is a shorthand property for setting 'margin-top', 'margin-right' 'margin-bottom' and 'margin-left' at the same place in the style sheet.

Borders and Padding

Borders and Padding act like table borders in plain html but can be applied to all elements. Borders are not well supported in all browsers, so you check them very carefully!

Just like in a table you should specify borders as well as padding. Borders will start where the element margins are. Padding will provide space between the boders and the content of the element.

'border-top-width' 'border-bottom-width' 'border-left-width' 'border-right-width'

The width of the keyword values are 'thin' 'medium' 'thick'.

'border-width'

Value:[thin | medium | thick

This property is a shorthand property for setting 'border-width-top', 'border-width-right', 'border-width-bottom' and 'border-width-left' at the same place in the style sheet.

H1 { border: solid thick red }

P { border: solid thick blue }

In the example above, 'H1' and 'P' elements will have the same border width regardless of font size. To achieve relative widths, the 'em' unit can be used:

H1 { border: solid 0.5em }

'border-color'

The 'border-color' property sets the color of the four borders. 'border-color' can have from one to four values, and the values are set on the different sides as for 'border-width' above. If no color value is specified, the value of the 'color' property of the element itself will take its place

'border-style'

Value: none | dotted | dashed | solid | double | groove | ridge | inset | outset

The 'border-style' property sets the style of the four borders. It can have from one to four values, and the values are set on the different sides as for 'border-width' above.

'padding-top' 'padding-right' 'padding-bottom' 'padding-left'

This property sets the left padding of an element.

BLOCKQUOTE { padding-left: 20% }

Padding values cannot be negative.

'padding'

Value:

The 'padding' property is a shorthand property for setting 'padding-top', 'padding-right' 'padding-bottom' and 'padding-left' at the same place in the style sheet. If four values are specified they apply to top, right, bottom and left respectively. If there is only one value, it applies to all sides.
